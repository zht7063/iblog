# Markdown 语法入门示例

Markdown 是一种轻量级标记语言，常用于写作、文档编写和网页内容排版。它语法简洁，易于学习。下面通过实际例子展示常用语法。

## 标题

使用 `#` 表示标题，`#` 越多层级越低：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 段落与换行

段落之间需空一行。  
在行尾加两个或以上空格可实现**强制换行**。

## 强调

- **加粗**：用双星号 `**文本**` 或双下划线 `__文本__`
- *斜体*：用单星号 `*文本*` 或单下划线 `_文本_`
- ~~删除线~~：用双波浪线 `~~文本~~`

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目（缩进两个空格）
```

### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 链接与图片

- 链接：`[百度](https://www.baidu.com)`
- 图片：`![替代文字](https://example.com/image.jpg)`

这里是一个[百度首页](https://www.baidu.com)的链接样例。

## 引用

> 这是一段引用内容。  
> 可以跨多行，用 `>` 开头。

## 代码

行内代码用反引号：`console.log("Hello")`

代码块用三个反引号包裹，并可指定语言：

```python
def hello():
    print("Hello, Markdown!")
```

## 表格

表格通过竖线 `|` 和连字符 `-` 构建：

```markdown
| 姓名   | 年龄 | 城市     |
|--------|------|----------|
| 张三   | 28   | 北京     |
| 李四   | 32   | 上海     |
```

效果如下：

| 姓名   | 年龄 | 城市     |
|--------|------|----------|
| 张三   | 28   | 北京     |
| 李四   | 32   | 上海     |
